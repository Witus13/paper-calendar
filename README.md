# paper-calendar

A useful calendar widget that you can use in various situations. Provides support for selecting dates, swiping between months and displaying events as dots below dates. All in Material Design style ðŸŽ¨

## Installation

Make sure you have installed NPM, then simply run  
`$ npm install --save @myfrom/paper-calendar`

Newer versions of this element work only with Polymer 3.  
For backwards compatibility use versions 0.1
